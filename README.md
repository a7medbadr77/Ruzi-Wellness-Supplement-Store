# Ruzi Wellness Supplement Store

A comprehensive e-commerce platform for ordering supplements, medical items, and cosmetics with online payment and delivery management in Rwanda.

## Features

✅ User Authentication (Login/Register)
✅ Admin Dashboard (Add/Remove/Update Products)
✅ Product Browsing by Category
✅ Shopping Cart Management
✅ Order Placement
✅ Payment Processing
✅ Delivery Address Management (Kigali, Rwanda)
✅ Order Tracking

## Tech Stack

- **Frontend**: React 18
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **Authentication**: JWT
- **Payment**: Stripe Integration
